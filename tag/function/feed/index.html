]]> In your code, add these lines

add_action('wp_ajax_FUNCTION-NAME', 'FUNCTION-NAME');
add_action('wp_ajax_nopriv_FUNCTION-NAME', 'FUNCTION-NAME');

function FUNCTION-NAME() { 
  echo "Its Working!"; 
}

There is no return value, instead print the values with echo. The url to call would be this:

http://testsite.com/wp-admin/admin-ajax.php?action=FUNCTION-NAME
