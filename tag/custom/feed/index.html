]]> Ever wondered how to create your own custom WordPress Shortcodes but can never find a simple explanation? Well look no further!

WordPress Shortcodes are text codes placed into the content of Pages and Posts that look something like this

[this_is_a_shortcode]

To create one, go to your WordPress theme through FTP. There, look for functions.php. At the bottom of the file, make sure you are typing in a PHP block, and put in a custom function.

After that is done, underneath add a line of code that looks like this

add_shortcode('this_is_a_shortcode', 'YOUR FUNCTION NAME');

This is what creates the Shortcode. The first variable in the function call is what the text found in between the square brackets when trying to call it. The second variable is making reference to which function to call when the Shortcode is called.
